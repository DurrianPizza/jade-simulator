# 开玉模拟器 - SPEC

## 1. Concept & Vision

一个精美的网页开玉模拟器，还原游戏中的开玉魄体验。界面风格古风玄幻，交互流畅有仪式感。用户可以选择开一块或批量开玉，实时显示收益统计，体验开玉的刺激与博弈感。

## 2. Design Language

- **美学方向**: 古典玄幻风格，暗金色调，玉石质感
- **配色**:
  - 主色: `#1a1a2e` (深紫黑)
  - 次色: `#16213e` (深蓝黑)
  - 强调色: `#e6b800` (金)
  - 背景: `#0f0f1a` (极深紫)
  - 文字: `#f0e6d3` (米白)
  - 阳玉: `#ff6b35` (橙红)
  - 阴玉: `#4ecdc4` (青绿)
- **字体**: "Microsoft YaHei", sans-serif
- **动效**: 开玉时有发光震动效果，卡片翻转展示结果

## 4. Features & Interactions

### 核心功能

1. **开单块玉**: 点击按钮开1块玉，动画展示结果
2. **批量开玉**: 输入数量N，一次性开N块玉，快速展示结果
3. **实时统计**: 显示总成本、总收益、净收益、回本率

### 开玉逻辑

- 50%阳玉，50%阴玉
- 主属性随机（10种: 伤害、速度、法术伤害、狂暴等级、物理暴击等级、法术暴击等级、固伤暴击等级、封印命中等级、法术伤害结果、穿刺等级、治疗能力）
- 副属性组合随机（从表格中按概率分布抽取）
- 价格: 按表格价格，如果为空则10块回收

### 数据来源

从Excel中提取的概率表，包含:
- 玉类型（阳玉/阴玉）
- 主属性
- 副属性组合
- 类型（AAA/AAB/ABB/非标准）
- 价格

### 交互细节

- 点击开玉: 按钮有按下效果
- 开玉动画: 玉牌闪烁发光，然后翻转显示结果
- 结果展示: 卡片显示玉类型(颜色区分)、主属性、副属性组合、价格

## 5. Component Inventory

### 玉牌卡片 (JadeCard)
- 尺寸: 180px × 240px
- 阳玉: 橙色边框，左上角三角符号
- 阴玉: 青绿色边框，左上角三角符号
- 正面: 显示玉类型、主属性、副属性组合
- 背面/底部: 显示价格
- 状态: 默认、发光中、翻转中

### 控制面板
- 开1块按钮: 大按钮，金色边框
- 批量开玉: 输入框 + 按钮
- 重置按钮: 清空所有结果

### 统计面板
- 4个指标卡片横向排列
- 数字动画变化效果

## 6. Technical Approach

- **技术**: 纯HTML + CSS + JavaScript，单文件
- **数据**: 内嵌JS变量存储概率数据
- **随机算法**: 根据概率分布随机选择副属性组合
- **状态管理**: 简单的全局变量
